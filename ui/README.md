# UI Directory

Next.js frontend for Sovereign Agent Treasury

## Deploy to Vercel

```bash
cd ui
npm install
npm run build
```

## Environment Variables

Create `.env.local`:

```
NEXT_PUBLIC_WALLETCONNECT_PROJECT_ID=your_project_id
```

## Features

- Wallet connection (RainbowKit)
- Treasury dashboard
- Transaction history
- Governance voting
- Agent status monitoring
- Multi-chain support (Monad testnet)
